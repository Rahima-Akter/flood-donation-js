Important FAQ

What is the Document Object Model (DOM)?
DOM allows us to manipulate the contents, style, and change the structure of an HTML webpage dynamically through JavaScript.

How do you select an element from the DOM?

There are many ways to select an element from DOM.But some mostly used options are-
1. document.getElementById()
2. document.getElementsByClassName()
3. document.querySelector()

What is event delegation in the context of DOM, and why is it useful?

Event delegation is used for handling events.It allows us to handle multiple children at once through the parent container.It is useful because we need to write less codes and it is dynamic.

How do you manipulate an element's attributes and styles using the DOM?

We manipulate an elements attributes by using setAttribute() method,and for syling we can use classList-object.
